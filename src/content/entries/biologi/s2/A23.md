#### Inti Materi

- **Air** → Evaporasi/Transpirasi → Kondensasi → Presipitasi → run-off.

- **Karbon–Oksigen** → **fotosintesis** (serap CO₂) ⇄ **respirasi** (lepas CO₂); pembakaran fosil.

- **Nitrogen** → **Fiksasi** (N₂ diikat *Rhizobium*/petir) → **Nitrifikasi** (amonia→nitrit→**nitrat**, *Nitrobacter*) → **Denitrifikasi** (nitrat→N₂).

- **Fosfor** → tidak lewat atmosfer; pelapukan batuan → fosfat → tumbuhan → kembali ke tanah.
